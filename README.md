# EBYS/Belgenet Taşınabilir Kurulum Sistemi

Bu araç, MEB EBYS ve imza servislerinin kurulumunu kolaylaştırmak için geliştirilmiştir.

### ⚠️ Önemli Not (SmartScreen Uyarısı)
Program dijital olarak imzalanmadığı için Windows çalıştırırken "Bilgisayarınızı korudu" uyarısı verebilir. 
Bu durumda:
1. **Ek Bilgi** (More Info) yazısına tıklayın.
2. **Yine de Çalıştır** (Run Anyway) butonuna basın.

**Neden Bu Uyarı Çıkıyor?**
Program, tarayıcı ayarlarını (EBYS izinleri için) kayıt defteri üzerinden düzenler. Bu işlemler "bilinmeyen yayıncı" olarak yapıldığı için Windows uyarı verir.
